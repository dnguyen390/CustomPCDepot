$620 Mid Tier Gaming PC Build

Project Info

  • Status: SOLD
  • Client: Josue
  • Project Date: 11 October, 2020

Gaming Computer i5 3.5GHz, 16GB RAM, GTX 970, 2TB Hard Drive, Windows 10, Office 19

This was a mid tier gaming computer build that I put together for a client. The overall objective of this build was to have the performance similar to a $1100 new computer build at nearly half the price. This build has clean and modern aesthetics, great price/performance and value for money, along with easy future upgradability of the components over the lifetime of the computer. Below is a parts breakdown list of all the componets used in this project.


Parts List


  • CPU: Intel Core i5 7400 3.5 GHz Quad Core Processor (upgradable to either a 6th or 7th gen i7 CPU)
  • Motherboard: MSI H110M Gaming Motherboard LGA 1151
  • RAM: 16GB DDR4 2400 MHz (upgradable up to 32GB RAM)
  • Video Card: EVGA GTX 970 4GB Gaming Graphics Card
  • Storage: 2TB Seagate 7200 RPM HDD
  • Power Supply: Thermaltake 700W PSU
  • Case: Zalman Z9 Neo Performance
  • Operating System: Windows 10 Home
  • Additional Software: Microsoft Office 2019
